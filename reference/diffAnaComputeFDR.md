# Computes the FDR corresponding to the p-values of the differential analysis

This function returns the FDR corresponding to the p-values of the
differential analysis.

## Usage

``` r
diffAnaComputeFDR(adj.pvals)
```

## Arguments

- adj.pvals:

  The adjusted p-values of the differential analysis

## Value

The computed FDR value (floating number)
